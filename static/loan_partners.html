귀사님께 제안드립니다 — 전국 생활숙박시설 여러 단지, 다수의 실거래를 보유한 홈앤스테이입니다.

홈앤스테이는

  • 전국 생활숙박시설·분양형호텔·콘도의 실거래가를 지도에서 조회하는 서비스입니다.
  • 국토교통부 실거래가와 건축물대장을 연동해 전수 조사 마스터 DB로 검증합니다.
  • 매입·잔금 대출이 필요한 소유주·매수인과 금융 파트너를 연결합니다.

가입 전에 실제 서비스 화면을 먼저 둘러보셔도 좋습니다

담보대출 상담 리드 연결
건물상세 화면의 '금융' 카드에 노출되어 매입·잔금 대출이 필요한 소유주·매수인의 상담 문의가 연결됩니다.
🛡️
신뢰 기반 파트너 심사
금융감독원에 등록된 대출상담사만 심사 후 승인됩니다. 대출모집인 통합조회(loanconsultant.or.kr) 확인을 거쳐 등록됩니다.
🆓
등록비 없음
가입비·등록비 없이 신청만으로 시작할 수 있습니다.

등록 절차

  1. 신청서 작성 (대출모집인 등록번호 필수)
  2. 금융감독원 대출모집인 통합조회에서 등록 여부 확인 — 금감원 등록 대출상담사만 심사 후 승인
  3. 승인 완료 문자 안내 후 금융 파트너로 활동 시작

신청 후 금감원 등록 확인 및 관리자 승인 시 문자로 안내드립니다.
